# An implementation of the Steiner Symmetrization

I have done a simple implementation of the Steiner Symmetrization using voxel-based representations. The code is available on my [GitHub page](https://github.com/matias-math/Steiner-Symmetrization-).

This project was developed in the context of the Mathematics Seminar, a course in the Bachelor's Degree in Mathematics at the University of Coimbra.
